Conditional branch

verified · 28 checks

ifThen/elseThen compiles to a decision service task plus an exclusive gateway pair.

DSL
workflow<Order>("order-triage") {
    items(orders)
    ifThen({ it.total > 100 }) {
        task("manual-review") { review(it) }
    } elseThen {
        task("auto-approve") { approve(it) }
    }
}
