Conditional branch

verified · 28 checks

condition/otherwise compiles to a decision service task plus an exclusive gateway pair.

DSL
workflow<Order>("order-triage") {
    input(orders)
    condition({ it.total > 100 }) {
        task("manual-review") { review(it) }
    } otherwise {
        task("auto-approve") { approve(it) }
    }
}
